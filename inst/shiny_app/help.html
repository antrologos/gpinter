This application is an interface to the R package gpinter. It provides a set of tools to estimate and manipulate distributions of income and wealth based on tabulations of the data such as those provided by tax authorities. This program rely on the so-called “generalized Pareto interpolation” methods.

Functionalities

The application can be used for the following tasks:

  • Estimate an entire distribution based on the values of the quantiles and the top shares at a few points.
  • Estimate an entire distribution based on the values of the top shares at a few points.
  • Individualize the household distribution under equal sharing of income among spouses.
  • Add up two components of income (say, labor and capital) assuming that the dependance between those two variables is characterized by a Gumbel copula.
  • Merge the distribution between several countries.
  • Simulate a random sample from the estimated distribution.

Usage

The navigation bar at the top of the window gives you access to the different parts of the application.

You should start with the “Input data” panel. This is were you specify the data you want to process, and the type of processing you want. The input data must take the form of a set of properly formatted files (see Input data format for details). Each of them represent the tabulation for one distribution. They can be either in CSV or Excel (.xls or .xlsx) format. For Excel files, each sheet is read separately as if it were a distinct file. You can import you files into the application using the “Browse” button under “Import file(s)”.

Just underneath, you will find a green “Run” button. It will only be enabled once you have imported your input data.

Clicking that button will run the programs.
